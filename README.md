# Cultivating Intelligence: Virtual Sensors for Scalable Smart Farming

## Project Overview

This project uses AI-based virtual sensors and IoT devices for scalable smart farming.

## Features

- ESP32 sensor monitoring
- CatBoost AI prediction
- LightGBM model
- Streamlit dashboard
- Water optimization

## Technologies Used

- Python
- ESP32
- CatBoost
- Streamlit
- ThingSpeak

## Results

- CatBoost Accuracy: 92.77%
- LightGBM Accuracy: 90%

## SDG Impact

Supports sustainable agriculture and water conservation.
